## Requirements

- macOS 13.0 or later
- Xcode 14.3
- [Homebrew](https://brew.sh)

## Getting stared

```shell
make bootstrap
open Calculator.xcworkspace
```

## Running tests

```shell
ARCH=arm64 make test
```
